# Canvas 射击小游戏
本项目是为了巩固学习到的canvas知识和面向对象思想而制作的一个小游戏。从游戏入手是学习面向对象的好方法。

## 项目说明
- 概述：Canvas 射击小游戏要求玩家控制飞机发射子弹，消灭会移动的怪兽，如果全部消灭了则游戏成功，如果怪兽移动到底部则游戏失败。
- 操作: 方向键左、右移动飞机，空格键发射子弹。

## 项目结构
- **index.html**: 游戏页面
- **style.css**: 页面样式
- **js**: 页面涉及的所有 js 代码
  - **app.js**: 页面逻辑入口 js
- **img**: 存放游戏的图片素材
- **视觉稿**: 存放游戏的视觉稿
- **readme.md**: 项目说明文档
